## Installation
1. Install Node.js
2. Install this repository
3. Run `yarn install`
4. Run `yarn start`
5. Open http://localhost:3000/
6. Enjoy!

## Usage
1. Click on the number to convert.
2. Click on the base to convert to.
3. Click on the base to convert from.
4. Enjoy!
